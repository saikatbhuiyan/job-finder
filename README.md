# Jobify

### Run The App Locally
- Run Backend Server
  - rename `.env.template` to `.env` and setup values for - MONGO_URL, JWT_SECRET, JWT_LIFETIME
  - install dependencies
    ```sh
     npm install
    ```
   - run server
      ```sh
       npm run server
      ```

- Run Front-End
  - go to client dir
    ```sh
     npm install
    ```
  - install dependencies
    ```sh
     npm install
    ```
   - go back to home dir
   - run client
      ```sh
       npm run client
      ```
 - Run Client and Server with same command
      ```sh
       npm run start
      ```
